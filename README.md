# Electro-Hub-Sales-Analysis-Dashboard-Power-BI-
This project focuses on analyzing the sales performance of Electro Hub, an electronics retail company, using Power BI. The dashboard provides actionable insights into sales, profit, discounts, and customer behavior across different time periods and regions. It enables stakeholders to make data-driven decisions by visualizing trends, comparing metrics, and identifying top-performing products.

Key Features


Top/Bottom 5 Products Analysis

Identify top and bottom 5 products based on Sales, Profit, and Quantity Sold.



Sales Trends Over Time

Visualize sales patterns across daily, monthly, quarterly, and annual intervals.



Sales vs Profit Relationship

Explore correlation between Sales and Profit using scatter plots and trend lines.



Period Comparison

Compare Sales, Profit, and Quantity Sold between any two user-selected time periods.



Discount Analysis

Calculate average discount offered in each discount category.



Order Summary

Display total number of orders placed during the selected period.



Detailed Order-Level Insights

Show Sales, Profit, Discount, Net Sales, and other fields for each order with dynamic filters:

Product
Date
Customer ID
Promotion Category





Regional Sales Analysis

Visualize sales by different cities to identify high-performing regions.




Tools & Technologies

Power BI for data visualization and dashboard creation.
Data Source: Electro Hub sales dataset (CSV/Excel).
DAX for calculated measures and KPIs.


Dashboard Highlights

Interactive filters for time period, product category, and region.
Dynamic charts: bar charts, line charts, scatter plots, and maps.
KPIs for Total Sales, Total Profit, Total Orders, and Average Discount.


How to Use

Download the .pbix file from this repository.
Open in Power BI Desktop.
Connect to the provided dataset or replace with your own.
Use slicers and filters to explore insights.


Business Impact
This dashboard helps:

Identify best-selling and least-performing products.
Understand seasonal trends and optimize inventory.
Monitor profitability and discount strategies.
Improve regional marketing campaigns.
